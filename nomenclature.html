Nomenclature and Notation

Two level quantum systems are used as qubits usually with a zero state $\left| 0 \right>$ that represents the column vector $(1,0)$ and a one state $\left| 1 \right>$ that represents the column vector $(0,1)$.

A probability distribution is a finite set of real numbers $p_x$ such that $0 \leq p_x$ and $\sum_x p_x = 1$.
